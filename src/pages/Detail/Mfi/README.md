# 📘 **MFI — 專業交易系統級檢查表（Professional Grade）**

---

# **I. 市場環境評估（Regime Filter）**

### **1. 波動度與流動性確認**

* ☐ 標的成交量是否足夠？（MFI 對低量股容易失真）
* ☐ 最近 20 天成交量是否維持穩定（無異常跳量或量枯）？
* ☐ 波動度是否突然放大導致 MFI 過度敏感？
  → 若 ATR 快速擴張，須降低 MFI 訊號的可信度

### **2. 趨勢方向濾網**

* ☐ 長週期均線（20 / 50 / 100）是否呈現一致方向？
* ☐ 價格是否在趨勢中（MA 有斜率）還是盤整中？
  → MFI 在盤整區精準度較高
  → 趨勢行情中，極端值可能維持很久（overbought ≠ sell）

---

# **II. 入場條件（Entry Conditions）**

## **A. 多頭入場（Buy）**

* ☐ MFI < 20（超賣區）
* ☐ 且 MFI 出現「低點抬高」
* ☐ 價格未跌破前低（跌破＝刀子落下）
* ☐ 價格在關鍵支撐（MA20 / MA50 / 前低）企穩
* ☐ 成交量開始回升（不接受無量反彈）
* ☐ 若搭配 BB：價格在下軌附近｜BB 收縮（較佳）
* ☐ MFI + 價格 出現背離（price lower low / MFI higher low）
  → 這是專業級強訊號

## **B. 空頭入場（Sell / Short）**

* ☐ MFI > 80（超買區）
* ☐ MFI 出現「高點降低」
* ☐ 價格無法再創高（momentum fatigued）
* ☐ 出現長上影、吞噬 K 或假突破
* ☐ BB 上軌走平或開始收斂
* ☐ MFI 與價格出現 bearish divergence
  → 這是機構最常用的反轉警訊

## **C. 趨勢追蹤策略（Momentum MFI 用法）**

（專業交易系統常見，但散戶較少用）

* ☐ MFI > 50 且持續上升 → 趨勢動能強
* ☐ 價格回測 MA20 附近，MFI 仍維持 >50 → 回檔買點
* ☐ 若 MFI 進入 60–80 且無掉頭 → 不提早空

---

# **III. 風險控管（Risk Control）**

### **1. 停損設計**

* ☐ 多單停損：跌破反轉 K 低點
* ☐ 空單停損：突破反彈 K 高點
* ☐ MFI 背離失效（背離後價格依舊破低／創高）＝立即停損
* ☐ 停損不超過資金的 1–1.2%

### **2. 部位大小（Position Sizing）**

* ☐ 若 MFI 接近極端（<15 或 >85）→ 降低部位 20–30%（極端值振盪大）
* ☐ 若 MFI 與價格背離 → 可提高部位（訊號品質高）
* ☐ 單標的部位不超過總資金 15%

### **3. 噪音過濾（False Signal Filter）**

* ☐ MFI 單日急拉急跌不可直接視為訊號
  → 必須確認連續 2–3 根 K
* ☐ 低量股票禁用 MFI（雜訊極高）

---

# **IV. 出場規則（Exit Strategy）**

### **A. 多頭出場**

* ☐ MFI > 80 且開始掉頭（動能衰退）
* ☐ 價格偏離 MA20 過遠（>3%）
* ☐ 連續三根高檔小實體或上影線
* ☐ 價格跌破 MA20（先減碼）
* ☐ MFI 形成「高點下降」 → 全部出場

### **B. 空頭出場**

* ☐ MFI < 20 且開始上彎
* ☐ 大量回補下影線
* ☐ MA20 被突破（先回補 50%）
* ☐ MFI 出現 bullish divergence → 全部回補

### **C. 趨勢倦怠判斷**

* ☐ 價格創新高但 MFI 未同步 → 趨勢末端訊號
* ☐ MFI 長期在 60–80 高檔卻開始跌破 50 → 動能告終
* ☐ 成交量與 MFI 同步掉落 → 價格通常會回填

---

# **V. 系統化紀律（Professional Discipline）**

* ☐ 所有進場必須至少符合「5 個以上」勾選條件
* ☐ 價格未出現反轉 K，不因 MFI 低於 20 就抄底
* ☐ 價格未出現上軌假突破，不因 MFI > 80 就放空
* ☐ 價格創高 / 破低但 MFI 未同步 → 嚴格觀察背離
* ☐ 每週回檢：

  * 是否忽略量能
  * 是否誤判背離
  * 是否在極端值過度操作
  * 是否落入震盪行情中被 MFI 過度假訊號洗
