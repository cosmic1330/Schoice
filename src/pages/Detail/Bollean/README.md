# 📘 **布林通道 — 專業交易系統級檢查表（專業版）**

---

# **I. 市場環境（Macro & Regime Filter）**

### **1. 波動度 Regime**

* ☐ BB 寬度（Band Width）位於近期區間的哪個分位數？

  * ＞70% → 趨勢行情
  * 30%–70% → 雜訊區
  * ＜30% → 壓縮準備爆發
* ☐ ATR 與 BB 寬度同向擴張 → 方向行情可信度↑
* ☐ 成交量是否同步增加？（無量突破＝假訊號高風險）

### **2. 趨勢方向濾網**

* ☐ 中軌 MA（一般20MA）是否呈現明確斜率？

  * 上彎 → 偏多
  * 下彎 → 偏空
* ☐ 價格是否持續在中軌上方（多）或下方（空）？
* ☐ 是否連續「靠著上軌走」（strong up trend）或「靠著下軌走」？

---

# **II. 入場條件（Entry Conditions）**

### **A. 多頭策略**

* ☐ 價格觸及下軌並形成反轉 K（Pin bar / Engulfing）
* ☐ RSI 未破低點（避免刀子落下）
* ☐ 中軌未明顯下彎（避免逆勢抄底）
* ☐ 成交量未異常萎縮
* ☐ 價格重新站回中軌 → 入場確認
* ☐ 上軌與下軌間距無異常收窄（避免假反彈）

### **B. 空頭策略**

* ☐ 價格觸及上軌形成反轉 K
* ☐ 中軌呈現下彎
* ☐ 價格無力站上上軌（假突破排除）
* ☐ 回落跌破中軌 → 入場確認
* ☐ 成交量正常（避免無量下跌假訊號）

### **C. 突破策略（Squeeze Breakout）**

* ☐ Band Width 處於低分位（<30%）
* ☐ 價格突破上軌（多）或下軌（空）
* ☐ 突破棒成交量放大（≥近20日平均1.3倍）
* ☐ 突破後 3 根 K 未回到通道內（假突破排除）
* ☐ 中軌斜率開始轉向

---

# **III. 風險控管（Risk Control）**

### **1. 止損設計**

* ☐ 下軌策略 → 停損放在「反彈低點下方」
* ☐ 上軌策略 → 停損放在「反轉高點上方」
* ☐ 突破策略 → 停損放在中軌或突破 K 下方
* ☐ 停損幅度不超過資金 0.8%–1.5%

### **2. 部位大小**

* ☐ 根據 Band Width 大小調整部位

  * Band 寬 → 波動大 → 降低部位
  * Band 窄 → 波動小 → 可提高部位
* ☐ 單筆不得超過總資金 15%

### **3. 不追價原則**

* ☐ 若因突破當下未跟上 → 僅在回測中軌再評估
* ☐ 絕不在「靠著外軌飛行」時追高，除非是系統策略且有量化回測

---

# **IV. 出場邏輯（Exit）**

### **A. 多頭出場**

* ☐ 價格觸及上軌後連續無法創高
* ☐ 價格跌回中軌 → 部位減半
* ☐ 價格跌破中軌兩次 → 全部出場
* ☐ Band Width 開始收窄 → 趨勢衰減

### **B. 空頭出場**

* ☐ 價格觸及下軌後收腳反彈
* ☐ 價格突破中軌 → 部位減半
* ☐ 價格突破中軌二次 → 全部出場

### **C. 突破策略**

* ☐ 價格回到通道內 → 視為假突破
* ☐ Band 開始收窄 → 減碼
* ☐ 出現反轉形態（吞噬、針對 K） → 考慮落袋

---

# **V. 系統級紀律（Professional Protocol）**

* ☐ 每日固定時間回測 BB 寬度、趨勢方向
* ☐ 所有入場需符合至少「4 個以上」勾選條件
* ☐ 任何時候都不可因情緒調整 MA 或 BB 設定參數
* ☐ 每週回顧：

  * 是否有追高
  * 是否忽略中軌斜率
  * 是否進場過早（在軌道外沒有反轉 K）
* ☐ 價格連續 2 次假突破 → 系統停止交易該標的 3 天
