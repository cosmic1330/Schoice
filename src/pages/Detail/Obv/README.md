II. 進場檢查（Entry Conditions）
(調整：每個買/賣點新增 MA/RSI/BB 條件；強制計算同向率 ≥70% 才進場。R:R ≥2:1。示範：每點末尾附計算範例。)
A. 多頭入場（Buy）
✔ 買點 1：OBV 上升突破前高（量先價行）

☐ OBV 創新高 (> 前高 1%)
☐ 價格尚未創高，但 < 前高 3%
☐ MA：價格 > 5MA/10MA
☐ RSI(14) >50
☐ BB：價格觸下軌或突破中軌向上（寬度 >20%）
☐ 成交量 ≥20日平均 120%
同向確認：計算 OBV/MA/RSI/BB 多頭比例 ≥70%。
→ 強訊號；目標 R:R 2:1（停損前低 -1 ATR）。示範：OBV 多、MA 多、RSI >50 (多)、BB 突破 (多) → 4/4=100% ≥70% → 進場。若 RSI=45 (中性) → 3/4=75% 仍進場；若 RSI<50 (空) → 3/4=75% 但需再確認。

✔ 買點 2：量價背離（Bullish Divergence）

☐ 價格創低 (< 前低 2%)
☐ OBV 未創低 (抬高 ≥1%)
☐ MA：價格回測不破 MA20
☐ RSI >40 (動能支持)
☐ BB：價格在下軌附近 (超賣) + 反轉 K
☐ MFI(14) >40 (輔助量能)
同向確認：OBV/MA/RSI/BB 多頭比例 ≥70%。
→ 低接機會；日線確認 ≥2日不破低。示範：OBV 背離 (多)、MA 不破 (多)、RSI=42 (多)、BB 下軌 (多) → 4/4=100%。若 BB 中性 → 3/4=75% 進場；若 MA 破 (空) → 2/4=50% <70% → 禁止。

✔ 買點 3：OBV 趨勢確認

☐ OBV 連續上揚 (≥3 higher lows，斜率 >0)
☐ 價格回測不破 MA20/前低 (<5%)
☐ MA：價格 > MA20/50
☐ RSI >50
☐ BB：突破中軌向上 (量增)
☐ 量價同步 ≥80% (近5日)
同向確認：OBV/MA/RSI/BB 多頭比例 ≥70%。
→ 波段追蹤。示範：OBV 上揚 (多)、MA > (多)、RSI=55 (多)、BB 中性 → 3/4=75% ≥70% 進場。若 BB 觸上軌 (空) → 2/4=50% 禁止。

III. 風險控管（Risk Control）
(調整：整合 RSI/BB 於停損；部位大小依同向率調整。)
1. 停損條件

☐ OBV 與價格同步走強 (背離失效 >1%)
☐ OBV 失守趨勢線 (<2%)
☐ 價格破/破反轉 K ±1 ATR
☐ RSI 反轉 (e.g., 多單 RSI <30) 或 BB 極端 (寬度 >50%)
停損幅度：1 ATR（本金 0.8-1.2%）；若同向率 70-80% → 擴大 1.2 ATR。

2. 部位大小

☐ 同向率 90-100% → 部位 100% (風險 <2%)
☐ 同向率 70-89% → 部位 70%
☐ OBV 噪音大 (標準差 >20%) → 降低 30%
☐ 總風險：單筆 < 帳戶 2%，基於 ATR。
