### K線形態策略概述

作為資深財經專家，我將根據K線（Candlestick）形態的經典技術分析原則，條列出常見的買進做多（多頭進場）、賣出做空（空頭進場）、做多出場（多頭離場，通常基於看跌信號）和做空出場（空頭離場，通常基於看漲信號）的策略。這些形態需結合趨勢、成交量、支撐/阻力位等因素使用，不應單獨依賴，以避免假信號。以下以表格形式呈現主要形態，包含形態描述、確認條件及建議應用。

#### 1. 買進做多（多頭進場）形態
這些是看漲反轉或延續形態，適合在下跌趨勢末端或盤整區買進。

| K線形態          | 描述                                                                 | 確認條件                          | 策略應用                              |
|-------------------|----------------------------------------------------------------------|-----------------------------------|---------------------------------------|
| 錘頭線 (Hammer)  | 下影線長、上影線短或無，實體小（紅或綠），位於下跌趨勢底。                | 次日開高走高，成交量放大。        | 買進做多，止損設於錘頭低點下方。      |
| 倒錘頭線 (Inverted Hammer) | 上影線長、下影線短或無，實體小，位於下跌趨勢底。                     | 次日紅K確認，突破上影線高點。    | 潛在反轉，適合低位買進。              |
| 看漲吞沒 (Bullish Engulfing) | 第一根綠K，第二根紅K完全吞沒前一根實體。                             | 發生於下跌趨勢末，成交量增加。    | 強力買進信號，目標為近期高點。        |
| 晨星 (Morning Star) | 三根K線：綠K、空隙小星線、紅K填補空隙。                             | 第三根紅K至少填補一半第一根綠K。  | 底部反轉，買進後追蹤趨勢。            |
| 刺穿線 (Piercing Line) | 第一根長綠K，第二根紅K開低但收於前一根中間以上。                     | 成交量放大，確認突破前高。        | 適合盤整或輕微下跌後買進。            |

#### 2. 賣出做空（空頭進場）形態
這些是看跌反轉或延續形態，適合在上漲趨勢頂端或盤整區賣出做空。

| K線形態          | 描述                                                                 | 確認條件                          | 策略應用                              |
|-------------------|----------------------------------------------------------------------|-----------------------------------|---------------------------------------|
| 射擊之星 (Shooting Star) | 上影線長、下影線短或無，實體小，位於上漲趨勢頂。                     | 次日開低走低，成交量放大。        | 賣出做空，止損設於射擊高點上方。      |
| 吊人線 (Hanging Man) | 下影線長、上影線短或無，實體小（紅或綠），位於上漲趨勢頂。            | 次日綠K確認，跌破下影線低點。    | 頂部反轉，適合高位做空。              |
| 看跌吞沒 (Bearish Engulfing) | 第一根紅K，第二根綠K完全吞沒前一根實體。                             | 發生於上漲趨勢末，成交量增加。    | 強力賣出信號，目標為近期低點。        |
| 黄昏星 (Evening Star) | 三根K線：紅K、空隙小星線、綠K填補空隙。                             | 第三根綠K至少填補一半第一根紅K。  | 頂部反轉，做空後追蹤趨勢。            |
| 烏雲蓋頂 (Dark Cloud Cover) | 第一根長紅K，第二根綠K開高但收於前一根中間以下。                     | 成交量放大，確認跌破前低。        | 適合盤整或輕微上漲後做空。            |

#### 3. 做多出場（多頭離場）形態
持有做多部位時，遇看跌信號應考慮出場，以鎖定獲利或止損。這些通常與賣出做空形態重疊，但重點在於離場時機。

| K線形態          | 描述                                                                 | 確認條件                          | 策略應用                              |
|-------------------|----------------------------------------------------------------------|-----------------------------------|---------------------------------------|
| 看跌吞沒 (Bearish Engulfing) | 如上所述。                                                           | 多頭趨勢中出現，突破近期支撐。    | 立即出場做多，評估是否轉做空。        |
| 黄昏星 (Evening Star) | 如上所述。                                                           | 成交量萎縮或放大確認反轉。        | 出場後觀察是否形成新低。              |
| 射擊之星 (Shooting Star) | 如上所述。                                                           | 位於多頭趨勢高點。                | 獲利出場，止損若已觸發。              |
| 吊人線 (Hanging Man) | 如上所述。                                                           | 次日確認跌勢。                    | 保守出場，避免假突破。                |
| 烏雲蓋頂 (Dark Cloud Cover) | 如上所述。                                                           | 突破前一根紅K低點。               | 出場並設定 trailing stop。            |

#### 4. 做空出場（空頭離場）形態
持有做空部位時，遇看漲信號應考慮出場，以鎖定獲利或止損。這些通常與買進做多形態重疊，但重點在於離場時機。

| K線形態          | 描述                                                                 | 確認條件                          | 策略應用                              |
|-------------------|----------------------------------------------------------------------|-----------------------------------|---------------------------------------|
| 看漲吞沒 (Bullish Engulfing) | 如上所述。                                                           | 空頭趨勢中出現，突破近期阻力。    | 立即出場做空，評估是否轉做多。        |
| 晨星 (Morning Star) | 如上所述。                                                           | 成交量放大確認反轉。              | 出場後觀察是否形成新高。              |
| 錘頭線 (Hammer)  | 如上所述。                                                           | 位於空頭趨勢低點。                | 獲利出場，止損若已觸發。              |
| 倒錘頭線 (Inverted Hammer) | 如上所述。                                                           | 次日確認漲勢。                    | 保守出場，避免假突破。                |
| 刺穿線 (Piercing Line) | 如上所述。                                                           | 突破前一根綠K高點。               | 出場並設定 trailing stop。            |
