# 📌 **Ichimoku 終極系統 v3（超專業版）

Ichimoku Ultimate Buy Checklist v3**

# ✅ **A. 趨勢確認（大方向）**

### **1. K 線 vs 雲區（Trend Filter）**

* ☐ 價格 *必須* 站在雲上
* ☐ 不能在雲內（盤整）
* ☐ 不能在雲下（空頭）

→ **未站雲上＝直接排除**

### **2. 雲區顏色與結構（Kumo Quality）**

* ☐ 目前雲為綠（Senkou A > Senkou B）
* ☐ 雲區厚度足夠（避免薄雲假突破）
* ☐ 雲呈現上升角度，不是水平（避免盤整）

---

# ✅ **B. 進場訊號（買點核心）**

### **3. TK Cross（短中期動能）**

* ☐ Tenkan-sen 上穿 Kijun-sen → 黃金交叉
* ☐ TK Cross 發生位置在雲上或雲邊（強訊號）
* ☐ Kijun-sen 不可向下（避免逆勢交叉）

→ 若 TK Cross 發生在雲下 → **無效，排除**

### **4. Chikou Span（延遲線動能確認）**

* ☐ 滯後線在過去 K 線上方
* ☐ 近 26 根內無明顯阻力（不卡在密集 K 區）

→ 若滯後線在價格下方 → **訊號無效**

---

# ✅ **C. 未來趨勢（前瞻結構）**

### **5. Future Kumo（領先跨距）**

* ☐ 未來雲為綠
* ☐ 未來雲角度向上
* ☐ 沒有變薄
* ☐ 無即將 Kumo Twist 翻空風險

（若正在出現 **紅轉綠的 Twist** → **加分**）

---

# ✅ **D. 三役好轉（最強買訊條件）**

### **6. 三役好轉 → 完美模式（最強力）**

若以下三項同時成立 → **S++ 等級買點**：

* ☐ 價格突破雲（Price > Cloud）
* ☐ Tenkan > Kijun（黃金交叉）
* ☐ Chikou 在價格上方

→ 三役好轉 = **優先考慮滿倉級別（依個人風控）**

若只有 2 個成立 → 仍可進場，但應降低部位。

---

# ✅ **E. 成交量、假突破過濾**

### **7. Volume（量能驗證）**

* ☐ 突破時有放量
* ☐ 放量幅度不應只來自一根孤量
* ☐ 無量突破需特別保守（假突破機率高）

---

# ✅ **F. 大盤環境過濾（避免逆勢）**

### **8. 市場方向一致性**

* ☐ 大盤（例如加權、S&P500、Nasdaq）也在雲上
* ☐ 大盤無明顯轉空訊號
* ☐ 行情非震盪盤（雲平坦）

→ 個股強勢但大盤弱勢 = **大幅降低勝率**。

---

# ✅ **G. 個股體質（避免雷股）**

### **9. 基本面（僅做黑名單排除）**

* ☐ 無重大利空（財報爆雷、訴訟、減資）
* ☐ 產業非明顯衰退
* ☐ 無短期爆量隔日沖亂象

→ 技術面訊號再強，也不能買雷股。

---

# ✅ **H. 風險與部位控管**

### **10. 入場位置**

* ☐ 不追離雲過遠的 K（避免追高）
* ☐ 入場不應離 Kijun 太遠（乖離過大需等回測）

### **11. 停損（Stop Loss）**

* ☐ 初始停損可設：

  * 基準線（Kijun）下方
  * 或雲下緣
* ☐ 一旦跌回雲內 → 立即降低部位
* ☐ 跌破雲 → **無條件出場**

### **12. 風險承擔**

* ☐ 單檔不超過 5–10% 資金
* ☐ 若屬中線持股：每週複查結構
* ☐ 不加碼在雲下或滯後線轉弱後

---

# 🟩 **I. 時間週期策略（Multi-TF 驗證）**

### **13. 時間週期一致性（MTF）**

* ☐ 日線 = 多頭
* ☐ 週線 = 不可逆勢（至少不能空頭）

→ 最強組合：**日線三役好轉 + 週線站雲上**
→ 最爛組合：**日線強、週線空** → 反彈機率，非趨勢。

##  如何解讀「雲翻轉成多雲」
預示變盤日：雲層翻轉交叉的那一點（雲層最薄的地方），通常被視為「翻轉日」或趨勢轉折點。在那一天，市場原有的趨勢力量最弱，最容易發生反轉或突破。
「如果目前的價格動能持續，到了那一天，下方的支撐力道會大於上方的壓力，市場比較容易維持在多頭格局。」
實戰意義：交易者通常會觀察價格在接近那個「翻轉日」時的表現。如果價格在當天確實站穩在雲層之上，就是對多頭趨勢的強烈確認。
