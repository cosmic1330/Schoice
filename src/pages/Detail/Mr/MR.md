# MR 指標 (MACD + RSI) 背離偵測策略

## 1. 核心指標定義

- **RSI (14)**：相對強弱指標，使用標準 **14 期**，衡量價格動能強弱。
- **MACD OSC**：MACD 柱狀圖（Histogram），反映短期與中期動能變化（預設參數 12,26,9）。
- **MR 策略核心**：結合 RSI 動能與 MACD 趨勢確認，特別注重**背離**訊號作為高機率轉折點。

---

## 2. 背離類型與交易訊號

### 2.1 底背離 (Bullish Divergence) - 強力買入訊號

- **價格狀態**：價格出現**新低 (Lower Low)**。
- **RSI 狀態**：RSI 形成**較高低點 (Higher Low)**，未跟隨價格創新低。
- **MACD 輔助確認**：MACD 柱狀圖開始收斂或翻正（動能轉強）。
- **技術意義**：賣壓衰竭，多頭動能悄悄累積 → **底部反轉訊號**。
- **進場建議**：出現底背離 + MACD 柱狀圖翻正 → 分批進場。

### 2.2 頂背離 (Bearish Divergence) - 強力賣出訊號

- **價格狀態**：價格出現**新高 (Higher High)**。
- **RSI 狀態**：RSI 形成**較低高點 (Lower High)**，未跟隨價格創新高。
- **MACD 輔助確認**：MACD 柱狀圖開始收斂或翻負（動能轉弱）。
- **技術意義**：買盤力竭，空頭壓力逐漸增強 → **高點反轉訊號**。
- **出場建議**：出現頂背離 + MACD 柱狀圖翻負 → 分批獲利了結或減倉。

---

## 3. 超買超賣 + MR 綜合訊號

| 訊號類型         | RSI 狀態          | MACD 狀態               | 價格情境             | 操作建議                     |
|------------------|-------------------|-------------------------|----------------------|------------------------------|
| **超賣 + 底背離** | < 30             | 柱狀圖翻正 / 金叉      | 價格破底但 RSI 抬頭  | **強力買入**（黃金坑）       |
| **超買 + 頂背離** | > 70             | 柱狀圖翻負 / 死叉      | 價格創新高但 RSI 回落| **強力賣出 / 減倉**          |
| **RSI 站上 50**  | > 50 且上升       | MACD 在零軸上方        | 沿上升通道           | **持多 / 續抱**              |
| **RSI 跌破 50**  | < 50 且下降       | MACD 在零軸下方        | 沿下降通道           | **持空 / 觀望**              |
| **盤整無效區**   | 45~55 窄幅震盪    | MACD 接近零軸          | 無明顯方向           | **避免交易，等待突破**       |

---

## 4. 策略執行重點

1. **背離優先**：背離是 MR 策略中最強的轉折訊號，勝率高於單純超買超賣。
2. **MACD 確認**：單純 RSI 背離不夠，必須搭配 MACD 柱狀圖方向變化才能進場，降低假背離風險。
3. **多時間框架共振**：
   - **日線 (D1)** 背離 → 波段級反轉（最高價值）。
   - **15分 / 30分 / 1H** 背離 → 日內反彈或修正機會。
4. **風險管理**：
   - 嚴設停損（前低/前高或 ATR 倍數）。
   - 分批進出場，避免一次性全倉。
   - 背離後若價格未跟隨反轉（連續背離），應立即停損。

---

## 5. 無效訊號過濾

- RSI 在 **45~55** 之間窄幅震盪時，背離容易失效（盤整市）。
- 強勢趨勢中，RSI 可能長期停留在超買（>70）或超賣（<30）區，此時勿逆勢操作。
- 重大消息面事件前後，技術訊號參考價值降低。

---

## 6. 總結使用原則

- **底背離 + 超賣 + MACD 轉強** = 高品質多頭進場點。
- **頂背離 + 超買 + MACD 轉弱** = 高品質空頭 / 出場點。
- **趨勢延續**：RSI 站穩 50 以上 + MACD 正值區 → 偏多操作。
- **最終提醒**：背離不是 100% 準確，務必搭配**成交量、均線、價格型態** 綜合判斷，並嚴格執行風險控制。

此策略適合 **台股波段與日內短線** 使用，建議在 TradingView 上實際標註背離練習。
